# AIで日本株のトレード戦略を自動化したら、税引後+50%になった

## 人間はルールを守れない

株で勝つ方法はシンプルだ。安い時に買って、高い時に売る。割安な株が下がった時に拾って、戻ったら売る。

問題は、人間がこれをやると余計なことをする。

シグナルが出ていないのに「そろそろ底だろう」で買う。損切りラインに達しても「もう少し待てば」と祈る。月に1回しかチャンスがないのに、退屈に耐えられず無駄なトレードをする。

そもそも、株価が10%下がった時に冷静でいられる人間がどれだけいるか。アルゴリズムは「あ、ストップロスですね」と0.5秒で損切りする。人間は「いや、ここから戻るはず......」と3日悩んで、さらに5%下がってから泣きながら売る。

トレードにおいて感情は負債でしかない。眠くならない、焦らない、「今回だけは特別」と言わないアルゴリズムの方が上手いに決まっている。

少ないトレード。厳しいフィルター。機械的な損切り。人間が一番苦手な規律を、AIに丸投げしたらどうなるか。

もともとはTradingView MCPというツールをClaude Codeで試してみたかっただけだった。「AIからチャートを操作してバックテストできるの面白いな」くらいの気持ちで始めたのに、気づいたら銘柄選定のスコアリングを何度も書き直し、ダッシュボードのデザインにこだわり、配当の税引後計算まで実装していた。これが沼というやつか。

## 結果

200万円スタート、2025年、日本の税率（20.315%）適用後:

| | |
|---|---|
| 税引後リターン | **+50.7%** |
| 税引後利益 | **+101万円** |
| 勝率 | **89.7%**（26勝3敗） |
| 最大ドローダウン | **6.7%** |
| プラス銘柄 | **8/8**（全銘柄プラス） |

2025年は日本株全体が好調だったので、この数字をそのまま将来に期待するのは危険。ただ、ロジック自体は一貫している。

## やっていること

**銘柄選び:** 東証の約3,000銘柄から「現金 > 借金」かつ「黒字」の株を自動抽出。利益率・値上がり勢い・割安さでスコアリングして、上位8銘柄を選出。

**売買タイミング:** RSI（株価の「売られすぎ度」を測る指標）が閾値を超えたら買い。戻ったら売り。

**リスク管理:** 日経平均が弱い時はストップロスを浅くする（10% → 7%）。ただしエントリーは止めない。暴落時こそ割安株を拾うチャンスなので。

## 全自動

毎日引け後にデータ取得・シグナル更新。毎月銘柄入れ替え。全て無人運用。

朝ダッシュボードを見て、信号機が緑に光っていたら買い。光っていなければ何もしない。

## 一番の学び

日経平均をエントリーフィルターに使おうとして4回失敗した。「日経が上がっている時だけ買う」「暴落時は買わない」——全部リターンが悪化した。

暴落時こそ割安株を拾う最大のチャンス。それをフィルターで止めるのは、戦略の一番おいしいところを捨てていた。
