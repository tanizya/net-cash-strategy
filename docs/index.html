Strategy Overview V6 Final

Core Insight: RSI平均回帰戦略にとって、市場暴落はリスクではなく最大のチャンス。 NI225をエントリーフィルターではなく、ストップロスの動的引き締めに使うことで、リターンを維持しながらDDを大幅に削減。

Entry (エントリー)

  • RSI(14) が 35 を上抜けクロス
  • 終値でシグナル判定
  • 翌営業日の寄付きで約定
  • フィルターなし (暴落時もエントリー)

Exit (エグジット)

  • RSI が 65 以上に到達 (利確)
  • SMA(20) を下抜け (トレンド転換)
  • ストップロスに到達 (損切り)

NI225 Dynamic Stop

  • NI225 > SMA(50): ストップ 10%
  • NI225 < SMA(50): ストップ 7% (引き締め)
  • エントリーには影響なし

Position Sizing

  • 1トレード: 資金の50% (最大100万)
  • 同時保有: 最大2ポジション
  • 最大リスク: 資金の10%/トレード
  • 月次DD 15%で当月エントリー停止
